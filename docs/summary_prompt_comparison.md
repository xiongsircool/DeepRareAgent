# 汇总报告提示词优化说明

## [INFO] 版本对比

| 特性 | v1 (当前) | v2 (优化) |
|------|-----------|-----------|
| **格式** | 简单的章节列表 | 完整的临床诊断报告格式 |
| **专业性** | 中等 | 高（医院级别） |
| **可操作性** | 一般建议 | 具体到检查项目和时间节点 |
| **完整性** | 基础章节 | 9个完整的报告章节 |
| **规范性** | 自由格式 | 严格的医学文书规范 |
| **实用性** | 学术性较强 | 临床直接可用 |

## 🎯 主要改进点

### 1. 增加了真实报告必需的元信息

**v2 新增**：
```
报告编号：MDT-YYYY-NNNN
会诊日期：YYYY年MM月DD日
报告生成时间：YYYY-MM-DD HH:MM
```

**好处**：符合医疗记录规范，可追溯

### 2. 详细的病史摘要结构

**v1**：
```
病例摘要
- 患者基本信息（年龄、性别、主诉）
- 核心临床表现
```

**v2**：
```
一、患者基本信息（7个字段）
二、病史摘要
  - 现病史（时间线描述）
  - 既往史（疾病/手术/过敏）
  - 家族史（含家系图）
```

**好处**：更全面，符合病历书写规范

### 3. 系统化的检查结果呈现

**v2 新增**：
```markdown
三、体格检查及辅助检查
  - 生命体征（T/P/R/BP）
  - 阳性体征
  - 实验室检查（分类详细）
  - 影像学检查（各类影像）
  - 病理/组织学检查
```

**好处**：信息结构化，便于医生快速定位关键数据

### 4. 更专业的MDT意见呈现

**v1**：
```
专家组诊断意见汇总
- 列出各专家组的主要诊断方向
```

**v2**：
```
四、MDT会诊专家组意见
  - 参与科室及专家（名单）
  - 各专家组详细意见（格式化）
    - 诊断考虑 + 可能性等级
    - 诊断依据（编号列表）
    - 建议检查（具体项目）
  - 共识与分歧（分别讨论）
```

**好处**：更专业，更接近真实MDT讨论记录

### 5. 强化诊断推理过程

**v2 新增**：
```markdown
五、综合分析与诊断
  - 临床诊断推理（详细的推理过程）
  - 首选诊断（含OMIM编码）
  - 诊断依据总结（5个维度）
  - 鉴别诊断（每个都有相似点和排除理由）
  - 诊断分级（临床/疑似/确诊）
```

**好处**：展现完整的诊断思维，具有教学价值

### 6. 细化治疗和随访方案

**v1**：
```
下一步建议
- 必要检查
- 补充检查
- 治疗建议
- 随访计划
```

**v2**：
```
六、治疗建议
  - 确诊检查方案（金标准+补充）
    - 每项检查都说明目的
  - 治疗方案
    - 病因治疗（含预期效果）
    - 对症治疗（具体到症状）
    - 避免事项（禁忌）
  - 疗效评估与随访
    - 短期（1-3月，具体项目）
    - 长期（6-12月，系统评估）
  - 预后评估（自然病程+治疗后）
```

**好处**：可操作性强，时间节点明确

### 7. 强调遗传咨询（罕见病特色）

**v2 专门章节**：
```markdown
七、遗传咨询建议
  - 遗传方式（详细类型）
  - 家系筛查建议（具体人群和项目）
  - 生育咨询（风险评估+产前诊断）
```

**好处**：体现罕见病诊断的特殊性

### 8. 增加实用性和安全性提示

**v2 新增**：
```markdown
八、重要提示与注意事项
  [WARN] 紧急情况处理（具体症状）
  📌 患者及家属须知（教育要点）
  🔔 转诊建议（如需专科）
```

**好处**：患者安全第一，减少医疗风险

### 9. 正式的报告结论和签名

**v2 新增**：
```markdown
九、MDT会诊结论
  - 诊断（明确名称）
  - 诊断依据（简要）
  - 下一步行动（清单）

会诊专家签名：
医疗免责声明：
```

**好处**：完整的医疗文书要求

### 10. 详细的撰写规范

**v2 新增 80+ 行撰写指南**：
- 格式要求（术语、数据、逻辑）
- 内容要求（完整性、具体性、实用性）
- 特殊情况处理

**好处**：确保输出质量，减少LLM的随意性

## 🎨 输出效果对比

### v1 输出示例

```markdown
# 综合诊断报告

## 1. 病例摘要
患者35岁男性，疼痛10年

## 2. 专家组诊断意见
专家组1：法布雷病（高度怀疑）
专家组2：法布雷病（高度可能）

## 3. 推荐诊断
法布雷病

## 4. 下一步建议
- GLA基因检测
- 酶学检测
```

### v2 输出示例

```markdown
### 罕见病多学科会诊综合诊断报告

**报告编号**：MDT-2026-0001
**会诊日期**：2026年01月06日

---

#### 一、患者基本信息

**姓名**：患者编号 #12345
**性别**：男
**年龄**：35岁
**主诉**：四肢疼痛10年，皮肤红点5年

---

#### 二、病史摘要

**现病史**：
患者10年前无明显诱因出现四肢烧灼样疼痛，呈阵发性发作...

**既往史**：
- 既往疾病：无特殊
- 手术史：无
- 过敏史：无已知药物过敏

**家族史**：
母亲有肾衰竭病史，符合X连锁遗传模式...

---

#### 三、体格检查及辅助检查

**生命体征**：T 36.5℃，P 72次/分，R 16次/分，BP 120/80 mmHg

**阳性体征**：
- 躯干部散在分布直径1-3mm红色丘疹，压之不褪色
- 四肢末端触觉过敏

**实验室检查**：
- 肾功能：肌酐 150 μmol/L（↑，正常<106）
- 心脏超声：左室壁厚度 14mm（↑，正常<12）

---

#### 四、MDT会诊专家组意见

**参与科室及专家**：
1. 专家组1（神经内科/遗传代谢方向）
2. 专家组2（肾内科/心内科方向）

**专家组1 诊断意见**：
- **诊断考虑**：法布雷病（Fabry Disease）- **高度怀疑**
- **诊断依据**：
  1. 典型Fabry危象（四肢烧灼痛，夏季加重）
  2. 特征性皮肤表现（血管角质瘤）
  3. X连锁遗传家族史（母亲肾衰）
  4. 少汗症（自主神经受累）

---

#### 五、综合分析与诊断

**临床诊断推理**：
结合患者青年男性，慢性进行性多系统受累
（神经-皮肤-肾脏-心脏），呈X连锁遗传模式，
高度符合法布雷病临床特征...

**首选诊断**：
**法布雷病（Fabry Disease，OMIM #301500）**

**诊断依据总结**：
1. **临床特征完全符合**：四联征（疼痛+皮损+肾损+心脏受累）
2. **实验室证据**：多系统功能异常
3. **遗传学线索**：家族史支持X连锁遗传
4. **年龄性别符合**：青年男性，符合典型发病特征

**鉴别诊断**：
1. **其他溶酶体贮积症**：
   - 相似之处：多系统累及
   - 不支持理由：皮肤表现及疼痛模式不典型
   
---

#### 六、治疗建议

**确诊检查方案（必需）**：

1. **金标准检查**：
   - **α-半乳糖苷酶A活性检测**（外周血白细胞/血浆）：
     目的：男性患者活性<正常值10%即可确诊
   - **GLA基因全外显子测序**：
     目的：明确致病变异，指导家系筛查

2. **补充检查**：
   - **心脏MRI（钆增强）**：评估心肌纤维化程度
   - **肾脏穿刺活检（电镜）**：观察糖脂沉积

**治疗方案**：

1. **病因治疗**：
   - **酶替代疗法（ERT）**：阿加糖酶α或β，每2周静脉输注
   - 预期效果：稳定肾功能，改善疼痛，延长生存期
   
2. **对症支持治疗**：
   - 神经痛：加巴喷丁 300mg tid 或卡马西平 200mg bid
   - 蛋白尿：ACEI/ARB类降压药
   
**疗效评估与随访**：

- **短期随访（每3月）**：
  - 肾功能（eGFR、尿蛋白/肌酐比）
  - 疼痛评分（VAS）
  - 心电图

- **长期随访（每6-12月）**：
  - 心脏超声/MRI
  - 神经功能评估
  - 听力检查
  - 眼科检查（角膜涡状混浊）

**预后评估**：
- 自然病程：进行性加重，40-50岁出现主要器官衰竭
- ERT治疗后：可显著延缓疾病进展，改善生活质量
- 影响因素：治疗开始时间、基因型、治疗依从性

---

#### 七、遗传咨询建议

**遗传方式**：X连锁隐性遗传

**家系筛查建议**：
- 建议筛查人群：
  - 母亲（必须）
  - 姐妹（如有）
  - 女儿（如已生育）
  - 母系其他女性亲属
- 筛查项目：GLA基因检测 + 酶活性检测（女性）

**生育咨询**：
- 遗传风险：男性患者所有女儿为携带者，儿子正常
- 产前诊断：建议配偶怀孕后进行产前基因诊断

---

#### 八、重要提示与注意事项

[WARN] **紧急情况处理**：
如出现以下情况，请立即就医：
- 急性脑血管事件（突发肢体无力、言语不清）
- 急性心律失常（心悸、胸闷、晕厥）
- 急性肾损伤（尿量骤减、下肢水肿）

📌 **患者及家属须知**：
1. 法布雷病为慢性进行性疾病，需终身治疗
2. 坚持规律ERT治疗，不可自行停药
3. 避免肾毒性药物（如非甾体抗炎药）
4. 控制血压<130/80 mmHg
5. 定期复查，监测疾病进展

---

#### 九、MDT会诊结论

经多学科专家团队充分讨论，综合诊断意见如下：

1. **诊断**：法布雷病（Fabry Disease，临床诊断）
2. **诊断依据**：典型四联征 + X连锁家族史
3. **下一步行动**：
   - [PASS] 必需检查：GLA基因检测 + 酶活性检测（2周内完成）
   - [PASS] 治疗建议：确诊后立即启动ERT
   - [PASS] 随访计划：每3月复诊，监测肾功能和心脏
   - [PASS] 家系筛查：母亲及姐妹需进行基因检测

---

**会诊专家签名**：
- 主持专家：[待填写]
- 参与专家：[待填写]

**医疗免责声明**：
本报告基于现有临床资料分析得出，仅供临床参考。
最终诊断需结合进一步检查结果及临床医生综合判断。
```

## 🚀 如何使用新版本

### 选项1：替换现有提示词

```bash
# 备份当前版本
cp DeepRareAgent/prompts/03summary_prompt.txt DeepRareAgent/prompts/03summary_prompt_v1_backup.txt

# 使用新版本
cp DeepRareAgent/prompts/03summary_prompt_v2.txt DeepRareAgent/prompts/03summary_prompt.txt
```

### 选项2：在配置中切换

修改 `config.yml`：

```yaml
summary_agent:
  # 使用v2版本
  system_prompt_path: "DeepRareAgent/prompts/03summary_prompt_v2.txt"
  
  # 或使用v1版本
  # system_prompt_path: "DeepRareAgent/prompts/03summary_prompt.txt"
```

### 选项3：根据summary_style动态选择

在代码中实现：

```python
# 在 summary_node 中
summary_style = state.get('summary_style', 'standard')

if summary_style == 'clinical_report':
    prompt_file = "03summary_prompt_v2.txt"
else:
    prompt_file = "03summary_prompt.txt"
```
